h1. About

Edoweb2 is the successor of edoweb http://www.hbz-nrw.de/angebote/hosting/edoweb/

Releases

Not released yet! The current status of this project is "in development". Current Version is still SNAPSHOT-0.0.1.

Build

mvn clean install

Test

To make use of the integrationTests install a fedora-3.6 and make the edoweb2-api.war available e.g.

cd edoweb2-api mvn clean install war:war cp target/edoweb2-api.war $FEDORA_HOME/tomcat/webapps launch tests with

cd integrationTests mvn clean install
